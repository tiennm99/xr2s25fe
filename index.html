What this is

One person changed phones. The page documents what changed at the hardware level — display, chip, memory, camera array, charging, network, ecosystem, lifespan — and renders that documentation six different ways.

Every format shares the same factual specs, the same SVG geometry, the same dates. What changes is the document tradition each format borrows from — engineering datasheet, Apple developer docs, Apple newsroom, Apple keynote, Apple compare page, or a side-by-side diptych. Same facts, six voices.

Each format is a self-contained static HTML file. No build step. No framework. Open one and it works.
